Five-Agent Band Workflow

AgentResponsibility
IntakeCaptures objective, constraints, bounded task
PlanDefines measurable requirements, scope, risks
ResolutionProduces solution, assesses every requirement
Issue IsolationExplains failure, supplies focused retry instruction
FinalizingProduces and delivers the terminal result — workflow ends
Human → Intake → Plan → Resolution ├─ requirements met → Finalizing └─ requirements not met → Issue Isolation ↓ Resolution retry ↓ Finalizing
2 / 5

Structured Collaboration

  • Band carries the full accumulated packet at every handoff
  • Stage results, evidence, quality, uncertainty grow with each agent
  • Routing is deterministic — based on requirements_met
  • Issue Isolation preserves exact failure context for retry
  • Only one retry cycle is allowed — controlled recovery
  • Terminal outcomes: completed · resolved_after_isolation · blocked · failed

Band is the real collaboration layer — not a wrapper, notification system, or output channel.
